# Hesabdar

## Persian Offline Personal Accounting

Hesabdar is a completely offline personal accounting application written as a single HTML file.

The application runs entirely inside the web browser and does not require:

Internet connection
Installation
Registration
Server
Database

All financial data remains on the user's device.

## Features

- Personal finance dashboard
- Income, expense and payment tracking
- Persian (Jalali) calendar
- Monthly financial reports
- Category analysis
- Interactive charts
- CSV export for Excel
- JSON backup and restore
- Smart backup merging
- Multiple accounts
- Custom categories
- Fully offline
- Responsive interface


##Technology
HTML5
CSS3
Vanilla JavaScript

No external libraries are required.

##Data Storage

Data is stored locally using the browser's Local Storage.

No information is transmitted over the Internet.
